# 冠軍點樣贏 FirstFelix——唔係快,係識諗

*(2026-07-11 · `rl/compare_vs_ff.py`,120 局,冠軍 = `best_raw_net.pt` ·
英文版:`champion_vs_ff.md`;另有廣東話網頁版 artifact)*

## 方法

冠軍網絡(0 號位,greedy)對三個 FirstFelix 打 120 局。**每一個**出牌位
都用 `ff_decide` 問埋 FirstFelix:「你坐呢個位會打邊隻?」(一個全新嘅 FF
即場由呢手牌+牌河重新定目標)。兩邊答案唔同,就將兩隻牌擺落**同一批
32 個想像世界**用 NN rollout 計價(配對 CRN)。兩邊選擇喺計價之前已經定咗,
所以呢啲係**無偏估計**——冇贏家詛咒。

## 主要數字

| 指標 | 數值 |
|---|---|
| 局賬 | +$1.57/局(120 局;長期參考 +$2.13) |
| 食糊 / 出銃 | 20.8% / 11.7% |
| 出牌決定 | 2,443 個 |
| **同 FF 打同一隻嘅比例** | **40.9%** |
| 計咗價嘅分歧 | 1,443 個 |
| 每個分歧嘅價值 | **+$1.32 ± 0.21**(6.4σ) |

冠軍唔係「FF 執靚咗嘅版本」:59% 嘅出牌,兩個牌手諗緊唔同嘢。

## 最大發現:贏喺個腦,唔係贏喺對腳

將分歧按「冠軍嗰隻牌同 FF 嗰隻比起上嚟做咗乜」分類:

| 類型 | 比例 | 平均值 |
|---|---|---|
| 鬥快——冠軍嗰隻行前咗 | 19.3% | **+$0.33**(最細!) |
| 進章——同速,生章多啲 | 20.9% | +$0.73 |
| 牌形——同速,個叫靚啲 | 43.7% | +$1.27 |
| **故意行慢一步** | 16.0% | **+$3.40 ± 0.96**(最大!) |

冠軍對 FF 最值錢嘅行為係**揀慢一步嘅牌**。單純鬥快反而係最細嘅優勢:
FF 一條筋追目標,行速唔慢;贏佢嘅係判斷,唔係速度,爭成十倍。

### 點解「慢」反而贏錢:上聽數係計步器,唔係計錢機

「慢一步」嘅意思好具體:打完冠軍嗰隻牌之後,手牌**距離叫糊多咗一步**
(以最少換牌次數計)。喺計步器上面,冠軍係倒退咗。但打牌唔係鬥少步——
係鬥**食得出糊、唔出銃**。而個價錢正正係咁計:兩隻牌喺同一批 32 個世界
**打到落幕**,+$3.40 係**最終局賬**嘅差,唔係「進度」嘅差。
「慢」只係形容嗰一刻喺計步器上嘅樣。

計步器睇唔到嘅三樣嘢,喺呢 231 個「慢」案例入面全部現晒形:

1. **結構值錢過步數。**44% 嘅案例,FF 嗰隻「快」牌係**由對子/刻子度抽出嚟**
   ——用一件一物三用嘅積木(將眼、碰、後備)換一步。而冠軍嘅「慢」牌
   **95% 係孤章**:計步器上蝕一步,但手牌副骨架原封不動。FF 而家慳嗰一步,
   遲啲往往要用兩步(加埋全部彈性)還返。
2. **生章值錢過地圖距離。**差一步但食嗰啲牌死得七七八八,實際上比
   差兩步但章章生嘅手牌**更慢**。上聽數計嘅係公里;張牌牆找嘅係車程。
3. **食唔食得出、守唔守得住。**三番起糊之下,最快嗰個叫好多時值 $0——
   一個法律上食唔到嘅海市蜃樓;而慢嗰隻牌通常仲要安全啲。所以「慢」呢類
   喺尾局爆升:牆剩三十隻以下,每個決定平均 **+$7.68**(開局先 +$2.03)。

FF 用完美紀律優化緊個計步器;冠軍優化嘅係計步器背後嗰樣真嘢。
「慢」(+$3.40)同「快」(+$0.33)之間十倍嘅差距,
就係將代理指標當成目標嘅、量得出嘅代價。

## 愈近收工,差距愈大

| 階段 | n | 每個分歧平均 |
|---|---|---|
| 開局(牆 > 55) | 663 | +$0.87 |
| 中局 | 461 | +$1.07 |
| 尾局(牆 ≤ 30) | 319 | **+$2.60** |

FF 嗰個凍結咗嘅目標、零重新評估,尾局俾人收割得最甘。

## FF 最貴嘅死症:追花色追到拆自己對子

FF 第一隻牌就定目標(長嗰門清一色,或者三對起對對糊),揀花色時會
**避開上家嘅花色**,然後無條件打晒目標以外嘅牌,永不回頭。

數據度捉到嘅最貴後果:手牌 `二索二索八索八索 + 一隻孤章`,FF 條避讓規則
揀咗個自己冇乜牌嘅花色做目標——跟住照規矩**拆自己對子**去追。同一局尾聲
連續三個決定,每個估值 **+$117 至 +$120**:冠軍打孤章留兩對,FF 拆對。

分類對照表仲確認兩個規律:

- **防守交換**(20.6%):FF 打中章牌,冠軍改打幺九/字(+$0.75–1.35)
- **尊重牌形**(17.4%):FF 由未完成嘅搭子抽牌走,冠軍保住個搭子(+$1.94–3.26)

## 俾人學嘅四句

1. **咪認死一條路**——每摸幾隻牌重新問:個目標仲啱唔啱?
2. **快唔係最緊要**——識慢(+$3.40)值鬥快(+$0.33)十倍。
3. **唔好為追花色拆自己對子。**
4. **尾局先分勝負**——牆過三十隻後每個決定價值升三倍。

## 老實講

- `ff_decide` 嘅 FF 係即場重新定目標;真局入面嘅 FF 揸住第一圈嘅目標。
  呢度計嘅係 FF 嘅決策函數,包括佢個目標規則——正正就係佢拆對子嗰條。
- 一步收益加埋 +$15.8/局,但唔會累積成局賬(實測 +$1.57–2.13):
  打咗更好嘅一隻之後,兩條時間線已分岔。
- 120 局局賬有 ±$3.6 噪音;精確嘅係逐個決定嘅配對估計。
